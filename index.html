发表于 2022-12-05 更新于 2022-12-07 分类于 University

第一题

首先，对于迭代格式 \(x_{k+1} = cos x_k\)，由于对任意 \(x\) 都有 \(-1\leq x \leq 1\)，因此无论 \(x_0\) 取什么值，都有

\[ -1\leq x_1 \leq 1 \]

即迭代格式 \(x_{k+1}=cosx_k\) 的迭代区间一定在 \([-1, 1]\) 中。

接下来，由于 \(x\in [-1, 1]\)，因此

\[ -1 < 0 \leq cosx \leq cos1 < 1 \]

那么不动点迭代收敛的第一个条件满足。

另外，由于 \(\varphi'(x) = sinx\)，因此对任意 \(x\in[-1, 1]\)，都有

\[ |\varphi'(x)| = |sinx| \]

又因为

\[ 0 \leq |sinx| \leq sin1, \ x\in[-1, 1] \]

所以

\[ |\varphi'(x)| < 1 \]

那么必然存在 \(0\leq L < 1\) 满足对于任意 \(x\in [-1, 1]\) 有 \(|\varphi'(x)| \leq L\)，因此不动点迭代收敛性的第二个条件满足。

综上，迭代格式 \(x_{k+1}=cosx_k\) 对于任意 \(x_0\) 都能够满足收敛条件。

\[ \lim_{x\rightarrow0}\bar\varphi'(x) = 1-\frac{1}{3}\lambda \]

\[ \bar\varphi(x) = x - 3\frac{sinx^3}{3x^2cosx^3} \]

第五题

观察到对于题设线性方程组的第 \(i\) 行，总有以下方程成立

\[ sin\boldsymbol{x}[i] = 1+sin\boldsymbol{x}[i+1]+(n+i)\boldsymbol{x}[i+2]+cos\boldsymbol{x}[i+3]+cos\boldsymbol{x}[i+4] \]

其中 \(\boldsymbol{x}[i]\) 表示长度为 \(n\) 的序列 \(\boldsymbol{x}\) 的第 \(i\) 项，并且如果有 \(i+a > n\)，则 \(\boldsymbol{x}[i+a]=\boldsymbol{x}[mod(i+a, n)]\)。

注意到每一行方程中都有一项 \((n+i)\boldsymbol{x}[i+2]\)，而其他项均为系数为1的三角函数，常数项也均为1。这一项的系数特别大，而且在每一行中都会出现在不同的 \(\boldsymbol{x}[i]\) 前，这意味着如果要使方程组快速收敛，所有的 \(\boldsymbol{x}[i]\) 都应该是一个很接近于0的数。

先假设初值均置为0，那么在第一次迭代中，可以快速口算得到 \(\boldsymbol{x}[1]_1 = 3\)。显然，由于存在 \((n+i)\boldsymbol{x}[i+2]\) 项，这种初值设定会在第一次迭代就导致整个方程组的求解发散。

注意到，如果所有的 \(\boldsymbol{x}[i]\) 都是一个很接近于0的数，那么会有

\[ \begin{aligned} sin\boldsymbol{x}[i] - sin\boldsymbol{x}[i+1] \approx 0\\ cos\boldsymbol{x}[i+3]+cos\boldsymbol{x}[i+4] \approx 2 \end{aligned} \]

即

\[ (n+i)\boldsymbol{x}[i+2] \approx -3 \]

也就是说

\[ \boldsymbol{x}[i+2] \approx \frac{-3}{n+i} \]

这个结果与我们的假设自洽了，那么可以用上面的这个关系作为初值，进行 \(\boldsymbol{x}\) 序列的初始化。

牛顿法求解

下面建立上述非线性方程组的迭代格式，令

\[ f_i(\boldsymbol{x}) = sin\boldsymbol{x}[i] - (1+sin\boldsymbol{x}[i+1]+(n+i)\boldsymbol{x}[i+2]+cos\boldsymbol{x}[i+3]+cos\boldsymbol{x}[i+4]) \]

在点 \(\boldsymbol{x}\) 上做线性化，可以得到

\[ f_i(\boldsymbol{x}) = f_i(\boldsymbol{x}^{(k)}) + \sum_{j=i}^{i+4}\frac{\partial f_i(\boldsymbol{x})}{\partial x_j}|_{\boldsymbol{x} = \boldsymbol{x}^{(k)}}(x_j-x_j^{(k)}) \]

那么可以得到非线性方程组的迭代格式

\[ \boldsymbol{x}^{(k+1)} = \boldsymbol{x}^{(k)} - [F'(\boldsymbol{x}^{(k)})]^{-1}F(\boldsymbol{x}^{(k)}) \]

由于无论 \(n\) 取多大，每一步求偏导时只有5个未知数有效，因此求偏导的计算量是 \(O(n)\)，而非 \(O(n^2)\)，那么直接使用牛顿法进行求解是完全可以接受的。每一行方程都可以得到偏导如下

\[ \begin{aligned} \frac{\partial f_i(\boldsymbol{x})}{\partial x_{i}}|_{\boldsymbol{x} = \boldsymbol{x}^{(k)}} = cosx_i^{(k)}\\ \frac{\partial f_i(\boldsymbol{x})}{\partial x_{i+1}}|_{\boldsymbol{x} = \boldsymbol{x}^{(k)}} = -cosx_{i+1}^{(k)}\\ \frac{\partial f_i(\boldsymbol{x})}{\partial x_{i+2}}|_{\boldsymbol{x} = \boldsymbol{x}^{(k)}} = -(n+i)\\ \frac{\partial f_i(\boldsymbol{x})}{\partial x_{i+3}}|_{\boldsymbol{x} = \boldsymbol{x}^{(k)}} = sinx_{i+3}^{(k)}\\ \frac{\partial f_i(\boldsymbol{x})}{\partial x_{i+4}}|_{\boldsymbol{x} = \boldsymbol{x}^{(k)}} = sinx_{i+4}^{(k)}\\ \end{aligned} \]

什么是数据结构？

让我们来考虑这样一组数据：一个班级中所有同学的姓名、性别、学号形成的一张表格。

这样的一组数据可以划分为以下几个层次

  • 数据集
    • 人（同学）
      • 姓名
      • 性别
      • 学号

数据集层次是最大的一个层次，这指向一个完整的数据表，或是一个数据文件；人的层次是数据在数据集中最粗的一个分层，或者说是最大的一个单位；姓名、性别和学号等信息则是人这一层次数据的子项，隶属于某一个特定的人。

通过USB备份与恢复微信聊天记录

发表于 2022-11-24 分类于 technical issues

在电脑上备份和恢复微信聊天记录必须通过局域网，又占局域网带宽，速度也不是很快。实际上，可以通过开启手机热点中的通过USB共享网络，通过USB数据线模拟局域网连接，进行电脑与手机之间的通信。

不过需要注意，通过USB连接到手机热点之后，需要关闭电脑上的WLAN连接，否则微信可能不会走USB这一路局域网。经过实测，微信备份和恢复聊天记录的速度最高可以达到30MB/s。
